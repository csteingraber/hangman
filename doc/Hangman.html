    load() click to toggle source

    This asks the user if they want to load a previous game and loads in the old values of @computer from the saves.yaml file.

    # File hangman.rb, line 60
def load
  puts "Would you like to load your previous game? (y/n)"
  puts ""
  response = gets.strip.downcase
  puts ""
  if response == "y" && File.exist?("saves.yaml")
    save = File.read("saves.yaml")
    @computer = YAML::load(save)
    puts "Your game has been loaded!"
    puts ""
  else
    puts "New game created!"
    puts ""
  end
end

    This asks the user if they want to save the game and will write the contents of @computer to YAML.

    # File hangman.rb, line 40
def save
  puts "Would you like to save your progress? (y/n)"
  puts ""
  response = gets.strip.downcase
  puts ""
  if response == "y"
    File.open("saves.yaml", "w") do |file|
      file.puts YAML::dump(@computer)
    end
    puts "Your game has been saved!"
    puts ""
  else
    puts "Lets just keep playing then!"
    puts ""
  end
end
    start() click to toggle source

    Starts a loop of player prompting and computer responses until the word is guessed or the hangman is drawn.

    # File hangman.rb, line 23
def start
  load
  until @computer.round_finished
    save
    @computer.respond_to_guess(@player.prompt)
  end

  if @computer.guessed
    puts "You won!"
  else
    puts "You lost!"
  end
end
